# Webシステム開発計画書

## 1. システムの対象
「第五人格」のハンター管理図鑑

## 2. ページ遷移
1. キャラクター一覧ページ（トップ）
   - 全キャラの名前と能力などを表形式で表示
2. キャラクター詳細ページ
   - 選択したキャラの外在人格、スキル詳細などを表示
3. 新規キャラ登録ページ
   - アップデートで追加された新キャラの情報を入力して保存する

## 3. データの項目（スキーマ）
- id: キャラクターID (数字)
- name: キャラクター名 (文字)
- skill: スキル（存在感ごとの寄託，再臨など）
- use: 特徴と立ち回り（全体負荷型，キャンプ型，チェイス型など）
- battle: サバイバー側からのキャラ対策
- identity: おすすめ人格
- characteristics: おすすめ特質
